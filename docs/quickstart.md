# Quickstart

## Install

Download a binary from GitHub Releases.

## Configure

`codemint --base-url https://app.codemint.app auth login`

Environment overrides:
- `CODEMINT_BASE_URL`
- `CODEMINT_PROFILE`

## Core commands

- `codemint auth login`
- `codemint auth whoami`
- `codemint items search --q "term"`
- `codemint org list`
